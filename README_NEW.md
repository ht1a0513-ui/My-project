# Event Registration System

This project extends the initial React + Vite frontend into a fully functional registration experience with state management, event handling, controlled components, and form validation.

## New features
- Advanced registration form with student profile fields
- Controlled components with two-way data binding using React state
- Form validation for required fields, email, password strength, phone number, and password confirmation
- Success and error feedback for the registration experience
- Show/Hide password toggles for better usability
- Reset button to clear the form

## React concepts covered
- useState for managing form data and validation feedback
- onChange, onSubmit, and onClick event handling
- preventDefault for form submission control
- Controlled input components and dynamic UI updates
- Reusable components and modular CSS styling

## Run locally
```bash
npm install
npm run dev
```
